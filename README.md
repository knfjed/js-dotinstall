# オブジェクト、プロパティ、配列

オブジェクトとは、複数の値をひとまとめにした値のこと。

配列は、複数の値（要素といいました）をひとまとめにし、それぞれを番号で区別するというものでした。

一方、オブジェクトでは、ひとつひとつの値が番号ではなく名前を持っています。

オブジェクトが中に持っている値のことをプロパティといいます。
オブジェクト.プロパティ名 と書く。
